Instructions:

  • Step 1: To set the initial 2d grid, one option is to select its width and height via the respective input forms and then click on cells to toggle between the states alive/dead. A cell is highlighted in blue color to indicate that it is alive.

    Alternatively, it is possible to upload a *.txt file consisting of 0s and 1s to set the initial grid. The numbers should be separated by spaces. A new line indicates the end of a row in the input matrix. Each line should contain the same amount of numbers.

  • Step 2: After setting the number of iterations for which the simulation should run (n) and the speed of the intervals in ms, one should click on the Start button to watch the simulation.
